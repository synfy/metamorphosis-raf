Metamorphosis: Real and Fiction
About Metamorphosis

Metamorphosis is an Origins addon, which replaces the default origins with new ones, centered around a theme. You start as a baby, with more weaknesses than powers, but eventually you grow into your more powerful adult form. Each time you die, though, you start over with a new random origin, and you're back to being a baby again.

Inspired in part by Afterlife SMP, part of the X Life series

Depends on the Fabric API, Origins, and Pehkui.

About Real and Fiction

This is the second Metamorphosis mod, made for Minecraft version 1.19.*. It features cryptids, mythical creatures, and a few real creatures that are just weird enough that they could have also been fiction.

This mod was made primarily for the private server that my friends and I play on, so if you aren't on that server, then I'm not really sure how you found this. Feel free to play with this mod, though if you're going to add it to a modpack I'd appreciate a heads-up, use the Issues tab on GitHub for that.

Commands and Mechanics

The growth from the baby version to the adult version is controlled by XP. In order to grow up, the following condition must be met: player must have 45 levels. Once the condition is met, the player must sleep in a bed to trigger the growth.

This mod can be played with a pseudo-hardcore mode, where players only have ten lives. After their last death, players will be put into spectator mode and will no longer be able to interact with the world. This is controlled through the scoreboard, so it can be enabled by running this command in chat:

/scoreboard objectives add deathCount deathCount ["Deaths"]

Operators on a server, or individuals on single player with cheats enabled can reroll their origin to randomly reassign a new origin to themselves or others.

/reroll playerName
